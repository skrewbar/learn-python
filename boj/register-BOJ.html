1. 백준 회원가입

백준에 접속합니다.

백준 메뉴 우측 상단에서 회원가입 버튼을 통해 가입합니다.

2. 설정

로그인을 하면 위와 같이 메뉴가 바뀝니다.
설정을 클릭합니다.

왼쪽에 위와 같은 사이드바 메뉴가 나타납니다.

2-1. solved.ac 연동

설정 메뉴에서 solved.ac를 선택하고, 파란색 사용하기 버튼을 클릭합니다.

설정 메뉴에서 보기를 선택합니다.

위 사진이 제가 쓰고 있는 설정입니다.
똑같이 설정하는 것을 추천합니다. 설정 후, 맨 밑의 파란색 "변경"버튼을 꼭 눌러줘야 합니다.

2-2. 언어 설정

자신이 쓸 언어를 왼쪽으로, 쓰지 않을 언어를 오른쪽으로 드래그 하면 됩니다. 이번에도 맨 밑의 파란색 "저장"버튼을 꼭 눌러줘야 합니다.

2-3. 소스 코드 공개

저는 하남자라 소스 코드 공개 설정을 맞았을 때만 공개로 설정했습니다. 하고 싶은 대로 설정하면 됩니다.
